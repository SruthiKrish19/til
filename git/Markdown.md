# Markdown

## Create Notice Blocks in Markdowns
### Note block
>[!NOTE]
>This is a note block
### Tip block
>[!TIP]
>This is a tip block
### Warning block
>[!WARNING]
>This is a warning block
### Caution block
>[!CAUTION]
>This is a caution block
### Important block
>[!IMPORTANT]
>This is a important block
